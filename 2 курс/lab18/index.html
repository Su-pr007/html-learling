Лабораторная работа №18. Форматирование текста в CSS_управление интервалами



Расстояние между буквами: letter-spacing

Значение по умолчанию, не добавляет дополнительное пространство мужду буквамми и символами - normal.
Значение задаётся в единицах длины, положительные значения увеличивают промежутки между буквами и символами. Например, letter-spacing: 1em.
Значение задаётся в единицах длины, отрицательные значения уменьшают расстояние между буквами и символами. Например, letter-spacing: -1em.
Можно использовать, как относительные, так и точные единицы измерения. Например, letter-spacing: 5px.
Устанавливает значение свойства в значение по умолчанию.
Наследует значение свойства от родительского элемента.


Растояние между словами: word-spacing

Значение по умолчанию normal, равное 0.25em.
Значение задаётся в единицах длины, положительные значения увеличивают промежутки между словами. Например, word-spacing: 1em.
Значение задаётся в единицах длины. отрицательные значения уменьшают промежутки между словами. Например, word-spacing: -1em.
Можно использовать, как относительные, так и точные единица измерения. Например, word-spacing: 5px.
Устанавливает значение свойства в значение по умолчанию.
Наследует значение свойства от родительского элемента.


ПРОБЕЛЫ между словами и ПЕРЕНОСЫ строк внутри элемента: white-space

normal - значение по умолчанию. Между словами вставляется только по одному пробелу, дополнительные пробелы отбрасываются. Текст переносится только в случае необходимости.
nowrap - Запрещает переносы строк, за исключением применения < br>

Vivamus tristique bibendum odio ut condimentum. Curabitur nulla nisi, lobortis sit amet molestie nec, mollis et mauris.
Fusce posuere purus id pharetra posuere.
Quisque pulvinar tincidunt metus in mollis. Duis a dignissim nunc.
pre -
пробелы в тексте не игнорируются, браузер отображает дополнительные пробелы и переносы строк. "Я повествую только о своём: Что в жизни много разного, и в нём Мы усмотреть должны все краски бытия и быта, Чтоб не остаться у разбитого корыта." Омар Хайям
pre-wrap - позволяет сохранить все пробелы в текста, делая разрывы строк там, где это необходимо. Phasellus suscipit, purus eusodales suscipit, urna ligula consequat nisl, id tempus augue augue non massa. Sed auctor odio ex, vel eleifend neque interdum eget.
pre-line - Удаляет лишние пробелы, за исключением случаев. Phasellus suscipit, purus eu sodales suscipit, urna ligula consequat nisl, id tempus augue nono massa. Sed auctor odio ex, vel eleifend neque interdum eget.
initial Устанавливает значение свойства в значение по умолчанию.
inherit Наследует значение свойства от родительского элемента.
